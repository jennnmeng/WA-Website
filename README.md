# Website Template for "WA: The Essence of Japanese Design"
This is a template for the book "WA: The Essence of Japanese Design" transformed into an interactive website viewable on console. Current pages will show placeholder text and images taken from the book directly. 

# Built With
- HTML/CSS - used for layout and design
- Javascript and Greensock - used for text, image, and page animations

# Acknowledgement
This project was inspired by the book "WA: The Essence of Japanese Design" written by Rossella Menegazzo and Stefania Piotti. All rights for the images and text used belong to the author and publishing house.
